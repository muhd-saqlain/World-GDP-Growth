#  World GDP Growth Analysis

This project analyzes and compares the GDP growth of major economies — **USA, China, India, and Pakistan** — using World Bank data.

##  Objective
To explore how the GDP growth rates of these countries have evolved from 2000 to 2023 and visualize the trends using Python.

##  Dataset
**File Name:** `world_gdp_growth.csv`  
**Source:** World Bank (Indicator Code: `NY.GDP.MKTP.KD.ZG`)  
**Description:** Annual GDP growth (in %) for all countries.

##  Steps Performed
1. Load and clean the dataset  
2. Select countries of interest  
3. Transform the data for analysis  
4. Visualize GDP growth trends (2000–2023)  
5. Compare average GDP growth rates  

##  Tools Used
- **Python**   
- **Pandas** → Data cleaning & manipulation  
- **Matplotlib / Seaborn** → Data visualization
